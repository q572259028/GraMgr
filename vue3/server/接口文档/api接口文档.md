## 1 登录

- 请求路径：login
- 请求方法：get
- 请求参数

| 参数名   | 参数说明 | 备注     |
| -------- | -------- | -------- |
| username | 用户名   | 不能为空 |
| password | 密码     | 不能为空 |

- 响应参数

| 参数名   | 参数说明    | 备注            |
| -------- | ----------- | --------------- |
| type   | 用户类型     |  -1：登陆失败，0：学生，1：老师，2：课程负责人，3：专业负责人，4：导员               

/*
{
    "type": 1
}
*/
