## 1 个人毕业要求达成度查询

- 请求路径：checkgra
- 请求方法：get
- 请求参数

| 参数名   | 参数说明 | 备注     |
| -------- | -------- | -------- |
| id | 用户账号   | 不能为空 |

- 响应参数

| 参数名 (data[].)  | 参数说明    | 备注            |
| -------- | ----------- | --------------- 
| sid   | 用户账号     |      |
| gid   | 毕业要求指标点编号     |      |
| grade   | 指标点成绩     |      |
| context   | 指标点内容    |      |

/*
{
    "data": [
        {
            "sid": 10001,
            "gid": 1,
            "grade": 0.5,
            "context": "XXX"
        }
    ]
}
*/
