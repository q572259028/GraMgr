## 1 任课老师修改学生指标点评价值操作 
- 请求路径：updategra
- 请求方法：get
- 请求参数

| 参数名   | 参数说明 | 备注     |
| -------- | -------- | -------- |
| sid | 学生id   | 不能为空 |
| cid | 课程id   | 不能为空 |
| gid | 指标点id   | 不能为空 |
| value | 修改后的值   | 不能为空 |

- 响应参数

| 参数名   | 参数说明    | 备注            |
| -------- | ----------- | --------------- 
| data   | 值为1表示成功     |     |
/*
{
    "data": 1
}
*/




## 2 根据任课老师id显示其负责的课程 
- 请求路径：get_course_id
- 请求方法：get
- 请求参数

| 参数名   | 参数说明 | 备注     |
| -------- | -------- | -------- |
| id |老师id   | 不能为空 |


- 响应参数

| 参数名 (data[].)    | 参数说明    | 备注            |
| -------- | ----------- | --------------- 
| cname   | 课程名     |     |
| id   | 课程id     |     |
{
    "data": [
        {
            "cname": "a",
            "id": 1
        }
    ]
}
