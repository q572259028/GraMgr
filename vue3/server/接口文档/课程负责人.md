## 1 课程负责人显示报表
- 请求路径：getcourse
- 请求方法：get
- 请求参数

| 参数名   | 参数说明 | 备注     |
| -------- | -------- | -------- |
| cid | 课程id   | 不能为空 |
| gid | 指标点id   | 不能为空 |
| state | 状态值   | 1：已通过 2：未通过 3：待审核 |

- 响应参数

| 参数名（data[].）   | 参数说明    | 备注            |
| -------- | ----------- | --------------- 
| sname  | 学生名     |     |
| id  | 学生所在班级的id     |     |
| grade  | 评价值    |     |

/*
{
    "data": [
        {
            "sname": "a",
            "id": 1,
            "grade": 1
        }
    ]
}*/
